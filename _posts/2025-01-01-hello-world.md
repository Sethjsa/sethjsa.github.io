This is an example post. Delete or replace this file to write your own.

To write a new post, create a file in `_posts/` named `YYYY-MM-DD-your-title.md` with this front matter at the top:

```
---
layout: post
title: "Your title here"
date: YYYY-MM-DD
---
```

Then write normally in Markdown below the front matter. Headers, **bold**, *italic*, links, code blocks, and lists all work out of the box.
